Install Tailwind CSS

A guide to setting up Tailwind CSS version 3!

Step 0: Initialize Your Project

Start by initializing a new Node.js project in your working directory.

npm init -y

Step 1: Install Tailwind CSS

Run the following commands to install Tailwind CSS and initialize its configuration.

npm install -D tailwindcss@3
npx tailwindcss init

Step 2: Configure tailwind.config.js

Update your tailwind.config.js file to include HTML files in the content array.

content: ["*.html"],
