Install Tailwind CSS

A guide to setting up Tailwind CSS version 3!

Step 0: Initialize Your Project

Start by initializing a new Node.js project in your working directory.

npm init -y

Step 1: Install Tailwind CSS

Run the following commands to install Tailwind CSS and initialize its configuration.

npm install -D tailwindcss@3
npx tailwindcss init

Step 2: Configure tailwind.config.js

Update your tailwind.config.js file to include HTML files in the content array.

content: ["*.html"],

Step 3: Create Input CSS File

Create a file named src/input.css and add the following Tailwind directives.

@tailwind base;
@tailwind components;
@tailwind utilities;

Step 4: Link Output CSS

Include the generated src/output.css file in your HTML file's head section.

Step 5: Build and Watch

Run the Tailwind CLI to process your CSS and watch for changes.

npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
